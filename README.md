# CODEVEDX
Frontend Development Internship Projects - Task Manager
## 📌 Project Description
This is a responsive Task Manager (To-Do App) developed using HTML, CSS, and JavaScript. It helps users manage daily tasks efficiently.

## 🚀 Features
- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks (All, Completed, Pending, High Priority)
- Dark & Light Mode
- Responsive Design
- LocalStorage Support

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage

## 📂 Project Files
- index.html
- style.css
- script.js
